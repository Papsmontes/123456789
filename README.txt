Modificaciones al juego snake.py

Se agregaron las siguientes modificaciones
- Lista con string que indican colores en la línea 19.
- Variables que contienen numeros random para seleccionar colores
distintos de la lista de colores antes creada en las lineas 20 y 21.
- cambio en las lineas 67 y 69, antes establecían un color estatico, pero 
ahora seleccionan un color diferente cada vez que se arranca el juego gracias
a las modificaciones antes mencionadas.
